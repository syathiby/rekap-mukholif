DOKUMENTASI STRUKTUR FOLDER DAN FILE APLIKASI AsuhTrack
======================================================
Versi: 2.0 (Diperbarui otomatis — mencerminkan kondisi nyata proyek per 2026-06-04)
Aplikasi ini adalah sistem manajemen kedisiplinan santri Pesantren Syathiby, Bogor.
Dibangun dengan PHP prosedural, MySQL (via MySQLi), Bootstrap 5, dan SweetAlert2.
Cocok dipelajari oleh developer penerus atau yang mengembangkan menggunakan AI.
