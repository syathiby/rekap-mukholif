DOKUMENTASI STRUKTUR FOLDER DAN FILE APLIKASI
======================================================
Dokumen ini adalah panduan lengkap untuk memahami apa saja isi dari setiap folder dan file pada aplikasi AsuhTrack / Rekap Mukholif ini. 
Sangat cocok dipelajari oleh developer penerus atau yang mengembangkan aplikasi ini menggunakan bantuan AI.

+--- .env                              => (PENTING) File "Top Secret". Isinya variabel rahasia seperti Password Database, alamat URL website asli. JANGAN PERNAH di-upload ke internet/Github terbuka!
